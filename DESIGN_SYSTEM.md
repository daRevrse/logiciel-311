# 🎨 Design System – Muno
Plateforme de gestion municipale intelligente

---

# 🧠 1. Vision globale

Muno est une plateforme digitale destinée aux collectivités locales, permettant de :
- centraliser les signalements citoyens
- améliorer la prise de décision
- optimiser les opérations municipales

## 🎯 Objectif UX
- Simplicité extrême côté citoyen
- Clarté et efficacité côté administration
- Compréhension en moins de 3 secondes

---

# 🎨 2. Identité visuelle

## 🎯 Positionnement
Moderne • Institutionnel • Accessible • Data-driven

---

## 🎨 Palette de couleurs

### 🔵 Couleurs principales
- Bleu profond : `#1E3A5F` → confiance, institution
- Bleu primaire : `#2F6FED` → actions, boutons

### 🟢 Couleurs secondaires
- Vert principal : `#2BB673` → validation, succès
- Vert clair : `#6ED3A3` → appui citoyen

### 🟠 Couleur d'état
- Orange : `#F59E0B` → en cours

### ⚪ Neutres
- Fond : `#F5F7FA`
- Blanc : `#FFFFFF`
- Texte : `#6B7280`

---

## ✍️ Typographie

### Police recommandée
- Inter (principal)
- Poppins (alternative)

### Hiérarchie
- H1 : Bold
- H2 : Semi-bold
- Texte : Regular

---

# 🧩 3. Design System UI

## 🧱 Composants de base

### Cartes (Cards)
- Fond blanc
- Border radius : 8px – 12px
- Ombre légère
- Padding : 16px – 24px

---

### Boutons

#### Primaire
- Background : `#2F6FED`
- Texte : blanc
- Radius : 8px

#### Secondaire
- Border : bleu
- Fond : blanc

#### Action citoyenne (IMPORTANT)
- Vert `#2BB673`
- Icône 👍
- Texte : "Appuyer"

---

### Badges (statuts)

- 🔵 Nouveau → bleu
- 🟠 En cours → orange
- 🟢 Résolu → vert

---

# 🖥️ 4. Version Web (Admin – Mairie)

## 🎯 Objectif
Permettre une gestion rapide et claire des opérations municipales.

---

## 🧱 Layout

### Sidebar (gauche)
- Logo Muno
- Navigation :
  - Dashboard
  - Signalements
  - Interventions
  - Marchés
  - Voirie
  - Documents
  - Paramètres

---

### Header (haut)
- Titre page
- Barre de recherche
- Filtres
- Profil utilisateur

---

## 📊 Dashboard

### Widgets
- Nombre total de signalements
- Signalements en cours
- Résolus
- Graphique d’activité

---

## 📋 Liste des signalements

### Chaque item contient :
- Titre
- Localisation
- Statut (badge)
- Nombre d’appuis 👍 (MIS EN AVANT)
- Image
- Actions

---

## 🗺️ Carte
- Carte avec pins
- Synchronisée avec la liste

---

## 🎯 UX clé
👉 Priorisation visuelle par nombre d’appuis  
👉 Lecture rapide  
👉 Décision facilitée

# 📱 5. Version Mobile (Citoyen)

## 🎯 Objectif
Permettre à n’importe quel citoyen de signaler un problème en moins de 30 secondes.

---

## 🧭 Navigation

- Accueil
- Signalements
- Nouveau signalement (+ bouton flottant)
- Profil

---

## 📍 Écran : Nouveau signalement

### Sections :

#### 1. Localisation
- Carte mini
- Adresse détectée
- Bouton modifier

---

#### 2. Photo
- Zone upload
- Icône caméra
- Aperçu image

---

#### 3. Catégorie
- Chips sélectionnables :
  - Voirie
  - Éclairage
  - Eau
  - Déchets
  - Autre

---

#### 4. Description
- Champ texte simple

---

#### 5. Signalements similaires (CLÉ)
- Liste courte
- Affiche :
  - Titre
  - Distance
  - 👍 appuis
- Bouton : "Appuyer"

---

#### 6. Bouton principal
- "Soumettre"
- Pleine largeur
- Bleu

---

## 📋 Écran : Liste des signalements

- Cards simples
- Statut visible
- 👍 appuis visibles
- Bouton "Appuyer"

---

## 🎯 UX clé

- Interface ultra simple
- Zéro friction
- Lecture immédiate
